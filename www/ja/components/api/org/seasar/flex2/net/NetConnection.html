Classpublic class NetConnection
InheritanceNetConnection flash.net.NetConnection

NetConnectionは、flash.net.Connectionクラスに、サーバロジック呼び出し時のcallBack関数を追加したクラスです。 CallBack関数には以下のものがあります。
  • AppendToGatewayUrl
  • ReplaceGatewayUrl
